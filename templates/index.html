Setup Guide ▼

Option 1: Browser Source (Recommended)

Available in OBS Flatpak or OBS with browser plugin installed.

  1. In OBS, click + under Sources and select Browser
  2. Paste the overlay URL from above
  3. Set width to 340 and height to 900 (adjust to fit)
  4. Click OK — the overlay updates automatically via live connection

Auto-Refresh: Not needed — updates are pushed to the overlay in real-time.

Background: Use Dark Panel for a semi-transparent overlay box, or choose a chroma key colour and add a Colour Key filter in OBS.

Option 2: Window Capture (Fallback)

For OBS without browser source support (e.g. Debian OBS package).

  1. Open the overlay URL in a browser window and resize it to fit the overlay
  2. In OBS, click + under Sources and select Window Capture
  3. Select the browser window showing the overlay
  4. Use a Crop/Pad filter to trim any extra browser chrome

Auto-Refresh: Recommended — tick Auto-Refresh above (10s is a good default). The browser reloads itself at the set interval to pick up changes.

Background: Dark Panel works best. Chroma key is possible but requires careful OBS filter tuning — text can be harder to read.

Chroma Key Tips

  • Magenta is usually better than green — less likely to clash with Elden Ring's environments
  • In OBS, add an Effect Filter → Colour Key on the source
  • Set Similarity to around 300–400 and Smoothness to 50–80
  • If text looks faded, lower the Similarity value
